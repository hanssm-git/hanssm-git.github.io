# openclaw-gws

openclaw-gws is a private, personal application that allows its owner to access their own Google Gmail and Google Calendar data via the Google APIs.

## Purpose

This app is used exclusively by its owner (a single person) to read emails and calendar events for personal productivity purposes. It is not a commercial product and has no external users.

## Features

- Read Gmail inbox and messages
- Manage Gmail labels
- Read Google Calendar events

## Privacy

This application does not collect, store, or share any data with third parties. All data remains on the owners own devices.
